    Terminal Based Donald Trump Simulator

    Project maintained by alexfortis Hosted on GitHub Pages — Theme by mattgraham

    ^[Tt]r(ump_[Bb])?ash$

    Have you ever wondered what would happen if Donald Trump were a process running on your computer? Wonder no more!

    Imagine Donald Trump is stuck in your terminal, and all he can do is tweet at you. Your only way of communicating with him is by sending him signals.

    • Written at HackAE on November 5-6, 2016
    • Used C to write the actual Trump program, bash scripting for the UI
    • Zenity is required for the UI
      • Mac: download from homebrew
      • Linux: download from the package manager if it's not already installed
